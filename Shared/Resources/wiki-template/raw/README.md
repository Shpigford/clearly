# raw/

Drop source material here — articles, PDFs, clippings, transcripts, meeting
notes, anything you want ingested into the wiki.

The agent reads files in this folder but never edits them. Treat `raw/` as
immutable archival material: the original sources behind the wiki's claims.

Markdown files here are still indexed for search, but they don't show up in
the log timeline and aren't considered "wiki pages."
